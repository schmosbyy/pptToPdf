LibreOffice 24.8 Help

Insert

Opens a submenu to insert rows and columns.

To access this command...

From the menu bar:

Choose Table - Insert.

From the context menu:

Choose Insert.

From the tabbed interface:

Choose Table.

From toolbars:

Insert

From the sidebar:

On the Properties panel, choose Table.


Rows Above

Insert a row above the row where the cursor is currently placed.

Rows Below

Insert a row below the row where the cursor is currently placed.

Rows

Opens dialog box for inserting rows.

Columns Before

Insert a column before the column where the cursor is currently placed.

Columns After

Insert a column after the column where the cursor is currently placed.

Columns

Opens dialog box for inserting columns.
