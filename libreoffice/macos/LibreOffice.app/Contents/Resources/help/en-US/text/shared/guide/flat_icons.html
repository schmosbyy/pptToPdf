LibreOffice 24.8 Help

Changing Icon Size

You can change the icon view between small and large icons.

  1. Choose LibreOffice - PreferencesTools - Options - LibreOffice.

  2. On the View tab page, select the Toolbar icon size.

  3. Click OK.
