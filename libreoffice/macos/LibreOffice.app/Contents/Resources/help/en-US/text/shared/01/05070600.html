Align Bottom

Vertically aligns the bottom edges of the selected objects. If only one object is selected in Draw or Impress, the bottom edge of the object is aligned to the lower page margin.

To access this command...

From the menu bar:

Choose Format - Align Text - Bottom.

From the tabbed interface:

Choose Text (text selected).Home

From the sidebar:

Select Properties - AlignmentParagraph deck.

From toolbars:

Bottom


Objects are aligned to the bottom edge of the bottom most object in the selection. To align the individual objects in a group, choose Format - Group - Enter Group double-click to enter the group, select the objects, right-click, and then choose an alignment option.
