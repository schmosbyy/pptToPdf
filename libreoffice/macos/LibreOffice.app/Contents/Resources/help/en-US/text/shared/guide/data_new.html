Creating a New Database

  1. Choose File - New - Database.

    This opens the Database Wizard, where you create a new database file.

  2. In the Database Wizard, select the type of database, and select the option to open the Table Wizard as the next wizard.

    The Table Wizard helps you to add a table to the new database file.
