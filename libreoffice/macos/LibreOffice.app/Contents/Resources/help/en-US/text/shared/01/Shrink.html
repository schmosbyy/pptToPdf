Decrease Size

Reduces the font size of the selected text.

To access this command...

From the menu bar:

Choose Format - Text - Decrease Size.

From the tabbed interface:

Choose Text - Decrease Size.

Choose Home - Decrease Size.

From toolbars:

Decrease Size

From the sidebar:

Choose Properties - Character - Decrease Size.

From the keyboard:

CommandCtrl + [
