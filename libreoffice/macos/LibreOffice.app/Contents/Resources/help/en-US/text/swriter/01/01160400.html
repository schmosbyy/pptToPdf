LibreOffice 24.8 Help

AutoAbstract to Presentation

Opens the current document as a LibreOffice Impress presentation. The current document must contain at least one predefined heading paragraph style.

To access this command...

Choose File - Send - AutoAbstract to Presentation


Included Outline Levels

Enter the number of outline levels to include in the new presentation. For example, if you choose one level, only the paragraphs that follow the "Heading 1" paragraph style are included.

Subpoints per Level

Enter the number of paragraphs that you want to include below each outline level (heading).
