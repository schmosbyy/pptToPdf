Sheet Comment Menu

Opens a submenu with comments commands.

To access this command...

From the menu bar:

Choose Sheet - Cell Comments.


Comment

Inserts a comment around the selected text, presentation slide, drawing page or at the current spreadsheet cursor position.

Show Comment

Shows the comment of the cell.

Hide Comment

Hides the comment of the cell.

Delete Comment

Deletes the comment of the cell.

Delete All Comments

Deletes all cell notes of the current sheet.
