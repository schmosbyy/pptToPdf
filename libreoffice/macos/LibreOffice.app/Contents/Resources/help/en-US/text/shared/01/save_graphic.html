LibreOffice 24.8 Help

Save

Opens the Image Export dialog to export the selected image.

To access this command...

From the menu bar:

Choose Format - Image - Save.

From the context menu:

Choose Save.

From the tabbed interface:

On the Image tab, choose Save.

On the Image menu of the Image tab, choose Save.

From toolbars:

Save


The default file format for export is determined by the graphic internal file format. You can export the image using another file format, from the list in the File Type or Filter drop-down box.
