Align Vertical Center

Vertically centers the selected objects. If only one object is selected in Draw or Impress, the center of the object is aligned to the vertical center of the page.

To access this command...

From the menu bar:

Choose Format - Align Text - Centered.

From the tabbed interface:

Choose Text (text selected).Home

From the sidebar:

Select Properties - AlignmentParagraph deck.

From toolbars:

Centered


To align the individual objects in a group, choose Format - Group - Enter Group double-click to enter the group, select the objects, right-click, and then choose an alignment option.
