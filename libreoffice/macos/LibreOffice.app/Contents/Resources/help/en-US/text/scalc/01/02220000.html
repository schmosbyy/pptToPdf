LibreOffice 24.8 Help

Go to Sheet

Go to a specified sheet.

To access this command...

From the menu bar:

Choose Sheet - Navigate - Go To Sheet.


Type a sheet name

Type some characters contained in the searched sheet name. List of sheets will be limited to the sheet names containing these characters. Search is case-sensitive. If empty, all visible sheets are listed.

Sheets

Lists the sheets in the current document. Hidden sheets are not listed. To select a sheet, press the up or down arrow keys to move to a sheet in the list. Double-clicking on a name will directly jump to this sheet.
