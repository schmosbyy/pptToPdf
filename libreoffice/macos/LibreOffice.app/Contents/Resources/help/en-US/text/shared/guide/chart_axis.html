Editing Chart Axes

To edit the axes of a chart that you have inserted:

  1. Double-click on the chart.

    A gray border appears around the chart and the menu bar now contains commands for editing the objects in the chart.

  2. Choose Format - Axis, then select the axis (or axes) that you would like to edit. A dialog appears.

  3. Select from the available sections and make the required changes (for example, select the Scale tab if you want to modify the scale of the axis).

  4. Click OK. In your document, click outside the chart to exit chart editing mode.
