Fill Mode

Activates the Fill Mode in the Detective. The mouse pointer changes to a special symbol, and you can click any cell to see a trace to the precedent cell. To exit this mode, press Escape or click the Exit Fill Mode command in the context menu.

To access this command...

From the menu bar:

Choose Tools - Detective - Fill Mode.

From the tabbed interface:

Choose Tools - Fill Mode.

On the Tools menu of the Tools tab, choose Fill Mode.

From toolbars:

Fill Mode


The Fill Mode function is identical to the Trace Precedent command if you call this mode for the first time. Use the context menu to select further options for the Fill Mode and to exit this mode.
