Insert Mode

Displays the current insert mode. You can toggle between INSRT = insert and OVER = overwrite. This field is only active if the cursor is in the input line of the formula bar or in a cell.

Click in the field to toggle the modes (except in the LibreOffice Basic IDE, where only the Insert mode is active). If the cursor is positioned in a text document, you may also use the Insert key (if available on your keyboard) to toggle the modes.

Mode

Result

INSRT

In the insert mode, new text is inserted at the cursor position and the following text is shifted to the right. The cursor is displayed as a vertical line.

OVER

In the overwrite mode, any existing text is replaced by new text. The cursor is displayed as a thick vertical line.
