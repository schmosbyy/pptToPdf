Navigating Changes

This feature is Writer-specific.


There are two available commands to navigate changes in a Writer document:

  • Edit - Track Changes - Next: Jumps to and selects the next change in the document, if any.

  • Edit - Track Changes - Previous: Jumps to and selects the previous change in the document, if any.

Using these commands in conjunction with the Accept and Reject commands allows navigating, accepting and rejecting changes without invoking the Edit - Track Changes - Manage dialog.
