Optimal Row Height

Set row height for selected table rows so that each row has the same height as the row with the tallest content.

Determines the optimal row height for the selected rows. The optimal row height depends on the font size of the largest character in the row. You can use various units of measure.

Set row height for selected table rows so that each row has the same height as the row with the tallest content.

To access this command...

From the menu bar:

Choose Table - Size - Optimal Row Height.

Choose Format - Rows - Optimal Height.

Choose Format - Table - Optimal Row Height.

From the context menu:

Choose Size - Optimal Row Height.

Select row, choose Optimal Height.

From the tabbed interface:

Choose Table - Optimize Size - Optimal Row Height.

On the Table menu of the Table tab, choose Optimal Row Height.

Choose Layout - Row - Optimal Height.

Choose Home - Row - Optimal Height.

From toolbars:

Optimal Row Height

From the sidebar:

Choose Properties - Table - Optimal Row Height.

Row height can increase with this option, with the table always growing downward. The total table height is never reduced by this option.

This option is similar to first minimizing row height for selected rows by using Minimal Row Height and then distributing those rows by using Distribute Rows Evenly, except that additional height is added to each row if necessary to prevent the total table height from reducing.

Add

Sets additional spacing between the largest character in a row and the cell boundaries.

Default value

Restores the default value for the optimal row height.
