Drawing Styles

Show styles for formatting graphical elements, including text objects.

To access this command...

From the menu bar:

Choose Styles - Manage Styles.

Choose View - Styles.

From the menu bar:

Choose View - Styles.

Choose Format - Styles - Manage Styles.

From the menu bar:

Choose View - Styles.

Choose Format - Styles - Manage Styles.

From the menu bar:

Choose View - Styles.

Choose Styles - Manage Styles.

From toolbars:

Styles

From the keyboard:

Command + TF11

General

Set the options for the selected style.

Area (Background, Highlighting)

Set the fill options for the selected drawing object or document element.

Line

Set the formatting options for the selected line or the line that you want to draw. You can also add arrowheads to a line, or change chart symbols.

Dimensioning

Changes the length, measurement and guide properties of the selected dimension line.

Connectors

Sets the properties of a connector.

Alignment

Sets the alignment of the paragraph relative to the margins of page.

Asian Typography

Set the typographic options for cells or paragraphs in Asian language files. To enable Asian language support, choose Languages and Locales - General in the Options dialog box, and then select the Asian box in the Default Languages for Documents area. The Asian typography options are ignored in HTML documents.

Tabs

Set the position of a tab stop in a paragraph.

Highlighting

Set the highlighting color of the text object.

Text Animation

Adds an animation effect to the text in the selected drawing object.

Indents and Spacing

Sets the indenting and the spacing options for the paragraph.

Font Effects

Specify the font effects that you want to use.

CharactersFont

Specify the formatting and the font that you want to apply.

Transparency

Set the transparency options for the fill that you apply to the selected object.

Shadow

Add a shadow to the selected drawing object, and define the properties of the shadow.
