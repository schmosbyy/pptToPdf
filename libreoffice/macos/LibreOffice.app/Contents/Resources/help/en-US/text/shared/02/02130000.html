Decrease Indent

Reduces the left indent of the current paragraph or cell content and sets it to the previous default tab position.

If you previously increased the indentation for several collectively selected paragraphs, this command can decrease the indentation for all of the selected paragraphs.The cell content refers to the current value under Format - Cells - Alignment.

To access this command...

From the menu bar:

Choose Format - Spacing - Decrease Indent.

From the tabbed interface:

Choose Home - Decrease Indent.

From toolbars:

Decrease Indent

From the sidebar:

On the Alignment panel of the Properties deck, choose Decrease Indent.

On the Paragraph panel of the Properties deck, choose Decrease Indent.

If you click the Decrease Indent icon while holding down the CommandCtrl key, the indent for the selected paragraph is moved by the default tab stop that has been set under LibreOffice Writer - General in the Options dialog box.
