LibreOffice 24.8 Help

Underline

Underlines or removes underlining from the selected text.

To access this command...

From the menu bar:

Choose Format - Text - Underline.

Choose Format - Text - Double Underline.

From the tabbed interface:

Choose Text - Underline.

Choose Home - Underline.

From toolbars:

Underline

Double Underline

From the sidebar:

Choose Properties - Character - Underline.

From the keyboard:

CommandCtrl + D (Double underline).


If the cursor is not in a word, the new text that you enter is underlined.

Single Underline

Underlines the selected text with a single line.

Double Underline

Underlines the selected text with two lines.
