LibreOffice 24.8 Help

Share Spreadsheet

Opens the Share Document dialog where you can enable or disable collaborative sharing of the document.

Sharing a spreadsheet allows several users to open the same file for editing at the same time.

To access this command...

From the menu bar:

Choose Tools - Share Spreadsheet

From the tabbed interface:

Choose Review - Share Spreadsheet.

From toolbars:

Share Spreadsheet


The spreadsheet needs to be on a shared drive, server, website, or similar. Some conditions must be met on operating systems with user permission management:

  • The shared file needs to reside in a location which is accessible by all collaborators.

  • The file permissions for both the document and the corresponding lock file need to be set so that all collaborators can create, delete, and change the files.

In order to correctly identify the changes, each collaborator should enter their name in LibreOffice - PreferencesTools - Options - LibreOffice - User Data on the Menu bar.
