Sorting Data

  1. Click in a database range.

    If you select a range of cells, only these cells will get sorted. If you just click one cell without selecting, then the whole database range will get sorted.

  2. Choose Data - Sort.

    The range of cells that will get sorted is shown in inverted colors.

  3. Select the sort options that you want.

  4. Click OK.
