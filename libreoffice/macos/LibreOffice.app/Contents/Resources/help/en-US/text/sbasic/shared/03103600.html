TypeName Function; VarType Function

Returns text or a numeric value that contain type information for a variable.

Syntax:


  TypeName (Varname As Variant) As String
  VarType (Varname As Variant) As Integer

Return value:

  • A text for TypeName suffixed with '()' for arrays.

  • An integer for VarType. The number 8192 is added to the returned value for arrays of type VarType.

Parameters:

Varname: The variable name that you want to determine the type of.
