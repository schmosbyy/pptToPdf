Rows Below

Insert a row below the row where the cursor is currently placed.

To access this command...

From the menu bar:

Choose Table - Insert - Rows Below.

From the menu bar:

Choose Sheet - Insert Rows - Rows Below.

From the context menu:

Choose Insert - Rows Below.

Select one or more rows, choose Insert Rows Below.

Choose Insert - Insert Row Below.

From the tabbed interface:

Choose Table - Insert Rows Below.

Choose Home - Insert Rows Below.

Choose Layout - Row - Insert Rows Below.

Choose Table - Insert Row Below.

From toolbars:

Insert Rows Below

From the sidebar:

Choose Properties - Table - Insert Rows Below.

Insert rows below the active cell. The number of rows inserted corresponds to the number of rows selected. If no row is selected, one row is inserted. The existing rows are moved downward.
