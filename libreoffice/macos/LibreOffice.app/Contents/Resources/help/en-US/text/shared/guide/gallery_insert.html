Inserting Objects From the Gallery

You can insert an object from the Gallery into a text, spreadsheet, drawing or presentation document by dragging and dropping or using the Context Menu. Inserting an object creates a copy of the object so any changes to the object in your document will not affect the object in the Gallery.

Inserting an object into a document

  1. Open the Gallery.

  2. Select a theme.

  3. Select an object using a single click.

  4. Drag the object into the document, or right-click to open the context menu and select Insert.

Replacing an object in a document

By dragging and dropping

  1. Open the Gallery.

  2. Select a theme.

  3. Select an object using a single click.

  4. Drag the object over the one you want to replace while holding down the CommandCtrl key.

Using the context menu

  1. Open the Gallery

  2. Select a theme.

  3. Left-click on the object you want to replace.

  4. Right-click on the object you want to replace it with and select Insert.

Inserting an object as a background graphic

  1. Open the Gallery.

  2. Select a theme.

  3. Select an object by a single click.

  4. Open the context menu and choose Insert - Background - Page or Insert - Background - Paragraph.

The option to insert an object as a background image is only available in LibreOffice Writer.
