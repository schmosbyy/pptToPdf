Adding Buttons to Toolbars

To make a button visible on a toolbar

Open the context menu of the toolbar (right click) and choose Visible Buttons and then select the button you want to display.

To add a button to a toolbar

  1. Choose Tools - Customize, and click on the Toolbars tab.

  2. In the Target box, select the toolbar you want to change.

  3. Select the command to be added in the Available Commands box. (Use the Category and/or Search boxes to restrict possibilities.)

  4. Click the Right Arrow icon to add the selected command.

  5. The check box in the Assigned Commands list controls whether the command is visible on the toolbar.

  6. You can rearrange the Assigned Commands list by selecting a command name and clicking Move Up and Move Down.

  7. Click OK.
