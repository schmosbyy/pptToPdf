LibreOffice 24.8 Help

Row

Sets the row height and hides or shows selected rows.

To access this command...

From the menu bar:

Choose Format - Rows.


Height

Changes the height of the current row, or the selected rows.

Optimal Row Height

Set row height for selected table rows so that each row has the same height as the row with the tallest content.

Determines the optimal row height for the selected rows. The optimal row height depends on the font size of the largest character in the row. You can use various units of measure.

Set row height for selected table rows so that each row has the same height as the row with the tallest content.

Hide

Hides selected rows or columns

Show

Choose this command to show previously hidden rows or columns.
