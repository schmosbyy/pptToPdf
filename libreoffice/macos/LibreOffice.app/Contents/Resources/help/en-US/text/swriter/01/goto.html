LibreOffice 24.8 Help

Go to Page

Opens a dialog box to enter which page number should be shown.

To access this command...

From the menu bar:

Choose Edit - Go to Page.

From the tabbed interface:

Choose Home tab menu - Go to Page.

From toolbars:

Go to Page

From the keyboard:

CommandCtrl + G

From the status bar:

Click on the Page number area.


Enter the page number or use the spin box to define the page to display.

The page number entered in the Go to Page dialog is absolute within the document and may not correspond to the page number displayed by the page numbering scheme.
