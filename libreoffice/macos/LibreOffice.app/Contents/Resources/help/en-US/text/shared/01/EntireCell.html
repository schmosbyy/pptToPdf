LibreOffice 24.8 Help

Cell

Selects the entire cell.

To access this command...

From the menu bar:

Choose Table - Select - Cell.

From the tabbed interface:

Choose Table - Select Cell.

From toolbars:

Select Cell

From the sidebar:

Choose Properties - Table - Cell.
