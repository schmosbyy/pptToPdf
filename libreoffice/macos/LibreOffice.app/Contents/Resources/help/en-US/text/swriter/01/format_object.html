Text Box and Shape

Opens a submenu to edit the properties of the selected textbox or shape.

To access this command...

From the menu bar:

Choose Format - Text Box and Shape.


Position and Size

Resizes, moves, rotates, or slants the selected object.

Line

Sets the formatting options for the selected line.

Area

Sets the fill properties of the selected drawing object.

Text Attributes

Sets the layout and anchoring properties for text in the selected drawing or text object.

Fontwork

Edits Fontwork effects of the selected object that has been created with the previous Fontwork dialog.
