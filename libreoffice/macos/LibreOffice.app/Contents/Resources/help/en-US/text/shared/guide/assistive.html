Assistive Tools in LibreOffice

LibreOffice supports some assistive technology tools like screen magnification software, screen readers, and on-screen keyboards.

A current list of supported assistive tools can be found on the Wiki at https://wiki.documentfoundation.org/Accessibility.


Supported Input Devices

LibreOffice provides the ability to use alternative input devices for access to all functions of LibreOffice.

  • Screen magnification software allow users with low vision to work in LibreOffice with caret and focus tracking.

  • On-screen keyboards enable users to perform almost all data input and commands with a mouse.

  • Screen readers allow visually impaired users to access LibreOffice with text-to-speech and Braille displays.
