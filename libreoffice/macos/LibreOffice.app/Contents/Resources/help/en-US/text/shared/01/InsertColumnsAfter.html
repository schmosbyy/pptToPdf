Columns After

Insert a column after the column where the cursor is currently placed.

To access this command...

From the menu bar:

Choose Table - Insert - Columns After.

From the menu bar:

Choose Sheet - Insert Columns - Columns After.

From the context menu:

Choose Insert - Columns After.

Select one or more columns, choose Insert Columns After.

Choose Insert - Insert Column After.

From the tabbed interface:

Choose Table - Insert Columns After.

Choose Home - Insert Columns After.

Choose Layout - Column - Insert Columns After.

Choose Table - Insert Column After.

From toolbars:

Insert Columns After

From the sidebar:

Choose Properties - Table - Insert Columns After.

Inserts columns to the right of the active cell. The number of columns inserted corresponds to the number of columns selected. If no column is selected, one column is inserted. The existing columns are moved to the right.
