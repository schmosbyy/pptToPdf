Using Content Controls in LibreOffice Writer

Content Controls

A content control is a placeholder text in which the current content will be replaced when clicking into the control. Content controls are individual controls to add and customize for use in templates, forms, and documents. Content controls can provide instructional text and set controls to disappear when users type in their own text.

Using Rich Text Content Control

  1. Place the cursor at the insertion location.

  2. Choose Form - Content Controls - Rich Text

  3. Choose Form - Content Controls - Properties.

  4. Optionally, select Content is placeholder text, add a Title and a Tag.

  5. Click OK to close the Content Control Properties dialog.

  6. To insert rich text, click in the control and type the text.

Using Plain Text Content Control

  1. Place the cursor at the insertion location.

  2. Choose Form - Content Controls - Plain Text.

  3. Choose Form - Content Controls - Properties.

  4. Optionally, select Content is placeholder text, add a Title and a Tag.

  5. Click OK to close the Content Control Properties dialog.

  6. To insert plain text, click in the control and type the text.

Using Picture Content Control

  1. Place the cursor at the insertion location.

  2. Choose Form - Content Controls - Picture.

  3. Click on the picture placeholder to open the Insert Image dialog, and select the image file. The image is anchored as-character in the document.

Using Check Box Content Control

  1. Place the cursor at the insertion location.

  2. Choose Form - Content Controls - Check Box.

  3. Optionally, select Content is placeholder text, add a Title and a Tag.

  4. Choose the checked and unchecked characters for the check box. Use the Select button to open the Special Character dialog and choose another character.

  5. Click OK to close the Content Control Properties dialog.

  6. Click on the check box to check or uncheck it.

Using Drop-down List Content Control

  1. Place the cursor at the insertion location.

  2. Choose Form - Content Controls - Drop-down List.

  3. Choose Form - Content Controls - Properties.

  4. Optionally, select Content is placeholder text, add a Title and a Tag.

  5. In the List Items box:

  • Use the Add button to add a new entry, entering the display name and the value of the item in the Content Control List Item Property dialog.

  • Use the Modify button to change the name and value pair of the selected entry.

  • Use the Remove button to delete the selected entry of the list.

  • Use the Move Up and Move Down buttons to position the selected entry in the list, respectively.

  1. Click OK to close the Content Control Properties dialog.

  2. To insert a content, click on the control to open the drop-down list, select one of the items displayed.

Using Combo Box List Content Control

Different from the Content Control Drop-down List, the Combo Box List allows you to directly insert text in the box, or choose one of the items in the list.

  1. Place the cursor at the insertion location.

  2. Choose Form - Content Controls - Combo Box.

  3. Choose Form - Content Controls - Properties.

  4. Follow instructions for the Content Control Drop-down List to insert display names and values to the list.

  5. To insert a content, click on the control to open a drop-down list, select one of the items displayed or type your custom text.

Using Date Content Control

  1. Place the cursor at the insertion location.

  2. Choose Form - Content Controls - Date.

  3. Choose Form - Content Controls - Properties.

  4. Choose one of the date formats in the Date Format list.

  5. Click OK to close the Content Control Properties dialog.

  6. To insert a date, click on the control to open a calendar object and choose the date.
