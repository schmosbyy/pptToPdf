LibreOffice 24.8 Help

Manage changes

Accept or reject recorded changes.

To access this command...

From the keyboard:

OptionAlt + 7

From the menu bar:

Choose Edit - Track Changes - Manage.

From the tabbed interface:

Choose Review - Manage.

In the Review menu of the Review tab, choose Manage.

From toolbars:

On the Track Changes toolbar, click

Manage Track Changes

From the sidebar:

Open the Manage Changes deck.


List

Accept or reject individual changes.

Filter

Set the criteria for filtering the list of changes on the List tab.
