Wrapping Text Around Objects

To Wrap Text Around an Object

  1. Select the object.

  2. Choose Format - Frame and Object - Properties, and then click the Wrap tab to choose the wrapping style that you want to apply.

    The current wrapping style is indicated by a bullet.

To Specify the Wrapping Properties

  1. Select the object.

  2. Choose Format - Frame and Object - Properties, and then click the Wrap tab.

  1. Set the options that you want.

  2. Click OK.

To Change the Wrapping Contour of a Graphic

You can change the shape that the text wraps around.

  1. Select the graphic, right-click, and then choose Wrap - Edit Contour.

  1. Use the tools to draw a new contour, and then click the Apply icon (green check mark).

  2. Close the Contour Editor window.
