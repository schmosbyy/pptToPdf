LibreOffice 24.8 Help

Select

Opens a menu for specific sheet selections.

To access this command...

From the menu bar:

Choose Edit - Select.

From toolbars:

Select


Select all Sheets

Selects all the sheets in your document.

Select Sheets

Opens the Select Sheets dialog to select multiple sheets.

Select to Next Sheet

Select to next sheet in the document.

Select to Previous Sheet

Select to previous sheet in the document.

Select Row

Select the entire row where the cursor is placed or the rows of the current cell selection.

Select Column

Select the entire column where the cursor is placed or the columns of the current cell selection.

Select Data Area

Selects the data area where the cursor or selection is located.

Select Unprotected Cells

Select the unprotected cells.

Select Visible Rows

Select the visible rows.

Select Visible Columns

Select the visible columns.
