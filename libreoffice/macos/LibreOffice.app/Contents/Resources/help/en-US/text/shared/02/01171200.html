Display Grid

Enables or disables the grid.

To access this command...

From the menu bar:

Choose View - Grid and Helplines - Display Grid.

From the context menu:

Grid and Helplines - Display Grid

From the context menu:

Grid and Helplines - Display Grid

From the tabbed interface:

Choose View - Display Grid.

From toolbars:

Display Grid
