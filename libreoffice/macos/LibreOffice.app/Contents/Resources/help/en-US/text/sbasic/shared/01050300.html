LibreOffice 24.8 Help

Manage Breakpoints

Specifies the options for breakpoints.

Breakpoints

Enter the line number for a new breakpoint, then click New.

Active

Activates or deactivates the current breakpoint.

Pass Count

Specify the number of loops to perform before the breakpoint takes effect.

New

Creates a breakpoint on the line number specified.

Delete

Deletes the selected breakpoint.
