Now Function

Returns the current system date and time as a Date value.

Syntax:


Now

Return value:

Date

Example:


  Sub ExampleNow
      MsgBox "It is now " & Now
  End Sub

The Now function measures time in seconds. To measure time in milliseconds use the Timer service.
