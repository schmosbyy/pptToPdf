LibreOffice 24.8 Help

Snap to Grid

Specifies whether to move frames, drawing elements, and controls only between grid points. To change the status of the snap grip only for the current action, drag an object while holding down the Control key Ctrl key.

Automatically aligns objects to vertical and horizontal grid lines.

To access this command...

From the menu bar:

Choose View - Grid and Helplines - Snap to Grid.

From the context menu:

Grid and Helplines - Snap to Grid

From the context menu:

Grid and Helplines - Snap to Grid

From the tabbed interface:

Choose View - Snap to Grid.

On the View menu, choose Snap to Grid.

From the tabbed interface:

Choose View - Snap to Grid.

On the View menu, choose Snap to Grid.

From the tabbed interface:

Choose Home - Snap to Grid.

On the View menu, choose Snap to Grid.

From toolbars:

Snap to Grid


To override this feature, hold down the OptionAlt key when you drag an object.
