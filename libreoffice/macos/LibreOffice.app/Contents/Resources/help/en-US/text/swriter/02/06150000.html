LibreOffice 24.8 Help

Add to List

Adds the current paragraph or selected paragraphs to the previous list.

To access this command...

From the menu bar:

Choose Format - Lists - Add to List.

From the context menu:

Choose List - Add to List.

From the tabbed interface:

On the Home menu of the Home tab, choose Add to List.

From toolbars:

Add to List
