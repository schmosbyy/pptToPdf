LibreOffice 24.8 Help

Scan

Inserts a scanned image into your document.

To insert a scanned image, the driver for your scanner must be installed. Under UNIX systems, install the SANE package found at http://www.sane-project.org. The SANE package must use the same libc as LibreOffice.

To access this command...

From the menu bar:

Choose Insert - Media - Scan.


Select Source

Selects the scanner that you want to use.

Request

Scans an image, and then inserts the result into the document. The scanning dialog is provided by the manufacturer of the scanner.
