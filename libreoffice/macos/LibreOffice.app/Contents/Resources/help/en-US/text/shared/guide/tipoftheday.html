LibreOffice 24.8 Help

Tip of the Day

The Tip of the Day dialog displays useful tips for the user.

To access this command...

Choose Help - Show Tip of the day.


The set of tips is collected from several LibreOffice community web pages, and is updated on each new release of the software.

The tip of the day is not specific to the current module.


Show tips on startup

Displays a dialog with a random tip on LibreOffice startup.

Next Tip

Displays another tip of the day in the same dialog.

OK

Close the Tip of the Day dialog.
