Columns Before

Insert a column before the column where the cursor is currently placed.

To access this command...

From the menu bar:

Choose Table - Insert - Columns Before.

From the menu bar:

Choose Sheet - Insert Columns - Columns Before.

Choose Sheet - Insert Cells - Entire column.

From the context menu:

Choose Insert - Columns Before.

Select one or more columns, choose Insert Columns Before.

Select one or more cells, choose Insert - Entire column.

Choose Insert - Insert Column Before.

From the tabbed interface:

Choose Table - Insert Columns Before.

Choose Home - Insert Columns Before.

Choose Layout - Column - Insert Columns Before.

Choose Table - Insert Column Before.

From toolbars:

Insert Columns Before

From the sidebar:

Choose Properties - Table - Insert Columns Before.

Inserts columns to the left of the active cell. The number of columns inserted corresponds to the number of columns selected. If no column is selected, one column is inserted. The existing columns are moved to the right.
