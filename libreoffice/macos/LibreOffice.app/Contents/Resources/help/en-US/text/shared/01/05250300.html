Send Backward

Moves the selected object down one level, so that it is closer to the bottom of the stacking order.

To access this command...

From the menu bar:

Choose Shape - Arrange - Send Backward.

From the context menu:

Choose Arrange - Send Backward.

From the keyboard:

CommandCtrl + minus sign.

From the tabbed interface:

Choose Image - Send Backward.

Choose Object - Send Backward.

Choose Home - Arrange - Send Backward.

From the menu bar:

Choose Shape - Arrange - Send Backward.

From the context menu:

Choose Arrange - Send Backward.

From the keyboard:

CommandCtrl + minus sign.

From the tabbed interface:

Choose Image - Send Backward.

Choose Object - Send Backward.

From the menu bar:

Choose Format - Arrange - Back One.

From the context menu:

Choose Arrange - Back One.

From the tabbed interface:

Choose Image - Back One.

Choose Object - Back One.

From toolbars:

Send Backward
