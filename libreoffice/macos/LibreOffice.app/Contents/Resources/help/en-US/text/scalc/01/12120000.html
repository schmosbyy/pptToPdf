LibreOffice 24.8 Help

Validity

Defines what data is valid for a selected cell or cell range.

To access this command...

From the menu bar:

Choose Data - Validity.

From the tabbed interface:

Choose Data - Validity.

On the Data menu of the Data tab, choose Validity.

From toolbars:

Data Validity


You can also insert a list box from the Controls toolbar and link the list box to a cell. This way you can specify the valid values on the Data page of the list box properties window.


Criteria

Specify the validation rules for the selected cell(s).

Input Help

Enter the message that you want to display when the cell or cell range is selected in the sheet.

Error Alert

Defines the error message that is displayed when invalid data is entered in a cell.

Reset

Resets changes made to the current tab to those applicable when this dialog was opened.
