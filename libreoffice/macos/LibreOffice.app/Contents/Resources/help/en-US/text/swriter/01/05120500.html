Delete

Deletes the selected column(s) from the table.

This command is only available if the cursor is in a table.

To access this command...

From the menu bar:

Choose Table - Delete - Columns.

From the context menu:

Choose Delete - Columns.

From the tabbed interface:

Choose Table - Delete Columns.

From the sidebar:

Choose Properties - Table - Delete Columns.

From the menu bar:

Choose Sheet - Delete Columns.

From the tabbed interface:

Choose Home - Column - Delete Columns.

Choose Layout - Column - Delete Columns.

From toolbars:

Delete Column
